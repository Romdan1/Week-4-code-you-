This repository contains solutions to various Python coding challenges. Each challenge is designed to practice fundamental programming concepts including loops, conditional statements, string manipulation, and working with lists.

Challenges Included
Reverse String
Function: reverse_string(text)
Reverses a given string using string slicing.
Palindrome Checker
Function: is_palindrome(text)
Checks if a string is a palindrome, ignoring spaces and case.
Find Maximum Number in a List
Function: find_max(numbers)
Finds the largest number in a list without using the built-in max() function.
Prime Number Checker
Function: is_prime(number)
Determines if a number is prime using loops and conditional logic.
Caesar Cipher
Function: caesar_cipher(text, shift)
Encrypts text by shifting letters by a given amount, preserving case and ignoring non-letter characters.
